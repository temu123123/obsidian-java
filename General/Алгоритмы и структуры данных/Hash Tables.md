#### Хэш-таблицы (Hash Tables)

- **Описание**: Хэш-таблица — это структура данных, которая обеспечивает доступ к элементам по ключу с использованием хэш-функции. В Java хэш-таблицы представлены классами `HashMap` и `Hashtable`.
- **Основные операции**: Вставка, удаление и поиск элементов по ключу.
- **Преимущества**: Быстрый доступ к элементам (O(1) в среднем случае).
- **Недостатки**: Возможны коллизии, которые требуют дополнительной обработки, случай ухудшенной производительности до O(n).
